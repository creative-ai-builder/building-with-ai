The ability to build is no longer limited to programmers.
Tomorrow's leaders won't just use AI — they'll know how to direct it.

This workshop is designed for curious middle schoolers who enjoy building, creating, problem-solving, space, technology, entrepreneurship, or simply exploring how things work. Over one week, students will learn how to work alongside modern AI tools to transform ideas into real software.

Week-Long Project

What Students Will Build

Students will build and deploy their own interactive Mars exploration application powered by real NASA mission data. On Friday, parents will be able to open and interact with software their child built during the week.

🗺️

Landing Site Maps

Interactive Mars landing site maps built with real mission data.

🤖

Mission Assistants

AI-powered Mars mission assistants that answer questions about space exploration.

📊

Mission Dashboards

Data visualizations and dashboards tracking real Mars mission telemetry.

🎨

Custom Interfaces

Student-designed, personalized interfaces that reflect their creative vision.

🚀

Exploration Missions

Student-created Mars exploration missions with goals, timelines, and science.

🌐

Live Web App

A fully deployed, shareable web application families can visit from any device.

Curriculum

What Students Will Learn

These are increasingly foundational skills in a world where AI is part of everyday work — skills that set students apart long after the week is over.

✦ Turn ideas into clear requirements
✦ Use AI as a creative and technical partner
✦ Build software through prompting
✦ Debug and improve AI-generated code
✦ Work with real-world data
✦ Test, iterate, and refine their work
✦ Present and explain their thinking
✦ Deploy software to the real internet
